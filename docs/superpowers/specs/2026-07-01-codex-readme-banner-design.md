# CODEX-баннер в README — дизайн

## Цель

Добавить заметную и честную атрибуцию OpenAI Codex в начало README без подмены авторов Git-коммитов.

## Вид

Сразу после заголовка `# DaVinci Console Bridge для Codex` размещается моноширинный ASCII-баннер выбранного пользователем варианта №1. Следующей строкой размещается текст `Assisted-by: OpenAI Codex`.

Баннер находится в fenced-блоке `text`, поэтому GitHub показывает его без изменения пробелов и выравнивания. Остальной README не редактируется.

## Проверка

Существующий контрактный тест README расширяется проверками текста `Assisted-by: OpenAI Codex` и характерной строки ASCII-баннера. Полный набор unittest должен пройти.
